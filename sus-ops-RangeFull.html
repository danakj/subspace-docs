Class sus::ops::RangeFull
class RangeFull
{ ... };

An unbounded range (..).

RangeFull is primarily used as a slicing index. It cannot serve as an Iterator because it doesn't have a starting point.

A RangeFull<usize> can be constructed as a literal as ".."_r.
