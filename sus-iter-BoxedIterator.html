Class sus::iter::BoxedIterator
class BoxedIterator
{ ... };

A BoxedIterator wraps another Iterator but pushes it onto the heap.

This makes the BoxedIterator itself be trivially relocatable, as it's just some pointers to the heap.

BoxedIterator is only constructible from an iterator that is not trivially relocatable.
