class MSGuidDecl

Defined at line 4200 of file /usr/lib/llvm-16/include/clang/AST/DeclCXX.h

A global _GUID constant. These are implicitly created by UuidAttrs.

struct _declspec(uuid("01234567-89ab-cdef-0123-456789abcdef")) X{};

X is a CXXRecordDecl that contains a UuidAttr that references the (unique) MSGuidDecl for the specified UUID.
