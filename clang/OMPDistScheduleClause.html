class OMPDistScheduleClause

Defined at line 6500 of file /usr/lib/llvm-16/include/clang/AST/OpenMPClause.h

This represents 'dist_schedule' clause in the '#pragma omp ...' directive.

In this example directive '#pragma omp distribute' has 'dist_schedule' clause with arguments 'static' and '3'.
