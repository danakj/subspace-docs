class OMPMapClause

Defined at line 5734 of file /usr/lib/llvm-16/include/clang/AST/OpenMPClause.h

This represents clause 'map' in the '#pragma omp ...' directives.

In this example directive '#pragma omp target' has clause 'map' with the variables 'a' and 'b'.

Inherits from OMPMappableExprListClause<OMPMapClause>, llvm::TrailingObjects<OMPMapClause, Expr *, ValueDecl *, unsigned int, OMPClauseMappableExprCommon::MappableComponent>
