class OMPDepobjClause

Defined at line 4701 of file /usr/lib/llvm-16/include/clang/AST/OpenMPClause.h

This represents implicit clause 'depobj' for the '#pragma omp depobj' directive. This clause does not exist by itself, it can be only as a part of 'omp depobj' directive. This clause is introduced to keep the original structure of class and its derivatives and to use the existing infrastructure of clauses with the list of variables.

In this example directive '#pragma omp depobj' has implicit clause 'depobj' with the depobj 'a'.
