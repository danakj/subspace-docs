class UsingPackDecl

Defined at line 3705 of file /usr/lib/llvm-16/include/clang/AST/DeclCXX.h

Represents a pack of using declarations that a single using-declarator pack-expanded into.

In the second case above, the UsingPackDecl will have the name 'operator T' (which contains an unexpanded pack), but the individual UsingDecls and UsingShadowDecls will have more reasonable names.
