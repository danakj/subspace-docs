class DecompositionDecl

Defined at line 4077 of file /usr/lib/llvm-16/include/clang/AST/DeclCXX.h

A decomposition declaration. For instance, given:

int n[3]; auto &[a, b, c] = n;

the second line declares a DecompositionDecl of type 'int (&)[3]', and three BindingDecls (named a, b, and c). An instance of this class is always unnamed, but behaves in almost all other respects like a VarDecl.
