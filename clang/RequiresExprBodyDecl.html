class RequiresExprBodyDecl

Defined at line 1973 of file /usr/lib/llvm-16/include/clang/AST/DeclCXX.h

This decl exists merely to serve as the DeclContext for the local parameters of the requires expression as well as other declarations inside it.

In this example, a RequiresExpr object will be generated for the expression, and a RequiresExprBodyDecl will be created to hold the parameter t and the template argument list imposed by the compound requirement.
