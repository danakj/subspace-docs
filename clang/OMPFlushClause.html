class OMPFlushClause

Defined at line 4624 of file /usr/lib/llvm-16/include/clang/AST/OpenMPClause.h

This represents implicit clause 'flush' for the '#pragma omp flush' directive. This clause does not exist by itself, it can be only as a part of 'omp flush' directive. This clause is introduced to keep the original structure of class and its derivatives and to use the existing infrastructure of clauses with the list of variables.

In this example directive '#pragma omp flush' has implicit clause 'flush' with the variables 'a' and 'b'.
