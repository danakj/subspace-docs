class QualifiedTemplateName

Defined at line 432 of file /usr/lib/llvm-16/include/clang/AST/TemplateName.h

Represents a template name that was expressed as a qualified name.

This kind of template name refers to a template name that was preceded by a nested name specifier, e.g., Here, the nested name specifier is "std::" and the template name is the declaration for "vector". The QualifiedTemplateName class is only used to provide "sugar" for template names that were expressed with a qualified name, and has no semantic meaning. In this manner, it is to TemplateName what ElaboratedType is to Type, providing extra syntactic sugar for downstream clients.
