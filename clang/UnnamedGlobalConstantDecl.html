class UnnamedGlobalConstantDecl

Defined at line 4257 of file /usr/lib/llvm-16/include/clang/AST/DeclCXX.h

An artificial decl, representing a global anonymous constant value which is uniquified by value within a translation unit.

These is currently only used to back the LValue returned by __builtin_source_location, but could potentially be used for other similar situations in the future.
