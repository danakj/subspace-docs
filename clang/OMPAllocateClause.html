class OMPAllocateClause

Defined at line 430 of file /usr/lib/llvm-16/include/clang/AST/OpenMPClause.h

This represents clause 'allocate' in the '#pragma omp ...' directives.

In this example directive '#pragma omp parallel' has clause 'private' and clause 'allocate' for the variable 'a'.
