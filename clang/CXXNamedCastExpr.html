Defined at line 372 of file /usr/lib/llvm-16/include/clang/AST/ExprCXX.h

Abstract class common to all of the C++ "named"/"keyword" casts.

This abstract class is inherited by all of the classes representing "named" casts: CXXStaticCastExpr for CXXDynamicCastExpr for CXXReinterpretCastExpr for reinterpret_cast, CXXConstCastExpr for and CXXAddrspaceCastExpr for addrspace_cast (in OpenCL).
