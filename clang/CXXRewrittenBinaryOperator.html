class CXXRewrittenBinaryOperator

Defined at line 283 of file /usr/lib/llvm-16/include/clang/AST/ExprCXX.h

A rewritten comparison expression that was originally written using operator syntax.

In C++20, the following rewrites are performed: - a == b -> b == a - a != b -> !(a == b) - a != b -> !(b == a) - For in - a @ b -> (a <=> b) @ 0 - a @ b -> 0 @ (b <=> a)

This expression provides access to both the original syntax and the rewritten expression.
