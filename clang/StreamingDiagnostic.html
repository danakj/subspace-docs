class StreamingDiagnostic

Defined at line 1110 of file /usr/lib/llvm-16/include/clang/Basic/Diagnostic.h

Basic

The streaming interface shared between DiagnosticBuilder and PartialDiagnostic. This class is not intended to be constructed directly but only as base class of DiagnosticBuilder and PartialDiagnostic builder.

Any new type of argument accepted by DiagnosticBuilder and PartialDiagnostic should be implemented as a '<<' operator of StreamingDiagnostic, e.g.

const StreamingDiagnostic& operator<<(const StreamingDiagnostic&, NewArgType);
