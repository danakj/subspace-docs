class NoInitExpr

Defined at line 5336 of file /usr/lib/llvm-16/include/clang/AST/Expr.h

Represents a place-holder for an object not to be initialized by anything.

This only makes sense when it appears as part of an updater of a DesignatedInitUpdateExpr (see below). The base expression of a DIUE initializes a big object, and the NoInitExpr's mark the spots within the big object not to be overwritten by the updater.
