class OMPFlushDirective

Defined at line 2736 of file /usr/lib/llvm-16/include/clang/AST/StmtOpenMP.h

This represents '#pragma omp flush' directive.

In this example directive '#pragma omp flush' has 2 arguments- variables 'a' and 'b'. 'omp flush' directive does not have clauses but have an optional list of variables to flush. This list of variables is stored within some fake clause FlushClause.
