class DesignatedInitUpdateExpr

Defined at line 5373 of file /usr/lib/llvm-16/include/clang/AST/Expr.h

In cases like: struct Q { int a, b, c; }; Q *getQ(); void foo() { struct A { Q q; } a = { *getQ(), .q.b = 3 }; }

We will have an InitListExpr for a, with type A, and then a DesignatedInitUpdateExpr for "a.q" with type Q. The "base" for this DIUE is the call expression *getQ(); the "updater" for the DIUE is ".q.b = 3"
