class ObjCObjectTypeLoc

Defined at line 940 of file /usr/lib/llvm-16/include/clang/AST/TypeLoc.h

A helper class for defining ObjC TypeLocs that can qualified with protocols.

TypeClass basically has to be either ObjCInterfaceType or ObjCObjectPointerType.
