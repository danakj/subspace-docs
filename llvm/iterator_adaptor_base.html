class iterator_adaptor_base

Defined at line 235 of file /usr/lib/llvm-16/include/llvm/ADT/iterator.h

CRTP base class for adapting an iterator to a different type.

This class can be used through CRTP to adapt one iterator into another. Typically this is done through providing in the derived class a custom implementation. Other methods can be overridden as well.
