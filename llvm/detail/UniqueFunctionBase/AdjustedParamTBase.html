struct AdjustedParamTBase

Defined at line 99 of file /usr/lib/llvm-16/include/llvm/ADT/FunctionExtras.h

Provide a type function to map parameters that won't observe extra copies or moves and which are small enough to likely pass in register to values and all other types to l-value reference types. We use this to compute the types used in our erased call utility to minimize copies and moves unless doing so would force things unnecessarily into memory.

The heuristic used is related to common ABI register passing conventions. It doesn't have to be exact though, and in one way it is more strict because we want to still be able to observe either moves *or* copies.
