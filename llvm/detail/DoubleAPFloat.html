class DoubleAPFloat

Defined at line 622 of file /usr/lib/llvm-16/include/llvm/ADT/APFloat.h

This mode implements more precise float in terms of two APFloats. The interface and layout is designed for arbitrary underlying semantics, though currently only PPCDoubleDouble semantics are supported, whose corresponding underlying semantics are IEEEdouble.
