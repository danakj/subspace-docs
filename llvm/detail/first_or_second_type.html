class first_or_second_type

Defined at line 1440 of file /usr/lib/llvm-16/include/llvm/ADT/STLExtras.h

Return a reference to the first or second member of a reference. Otherwise, return a copy of the member of a temporary.

When passing a range whose iterators return values instead of references, the reference must be dropped from `decltype((elt.first))`, which will always be a reference, to avoid returning a reference to a temporary.
