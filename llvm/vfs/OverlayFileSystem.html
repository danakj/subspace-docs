class OverlayFileSystem

Defined at line 366 of file /usr/lib/llvm-16/include/llvm/Support/VirtualFileSystem.h

A file system that allows overlaying one on top of another.

Consists of a stack of >=1 objects, which are treated as being one merged file system. When there is a directory that exists in more than one file system, the contains a directory containing the union of their contents. The attributes (permissions, etc.) of the top-most (most recently added) directory are used. When there is a file that exists in more than one file system, the file in the top-most file system overrides the other(s).
