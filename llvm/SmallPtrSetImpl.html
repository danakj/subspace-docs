class SmallPtrSetImpl

Defined at line 344 of file /usr/lib/llvm-16/include/llvm/ADT/SmallPtrSet.h

A templated base class for which provides the typesafe interface that is common across all small sizes.

This is particularly useful for passing around between interface boundaries to avoid encoding a particular small size in the interface boundary.
