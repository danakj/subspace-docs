class list_storage

Defined at line 1559 of file /usr/lib/llvm-16/include/llvm/Support/CommandLine.h

Define how to hold a class type object, such as a string. Originally this code inherited from std::vector. In transitioning to a new API for command line options we should change this. The new implementation of this list_storage specialization implements the minimum subset of the std::vector API required for all the current clients.
