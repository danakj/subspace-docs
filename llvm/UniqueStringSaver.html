class UniqueStringSaver

Defined at line 44 of file /usr/lib/llvm-16/include/llvm/Support/StringSaver.h

Saves strings in the provided stable storage and returns a StringRef with a stable character pointer. Saving the same string yields the same StringRef.

Compared to StringSaver, it does more work but avoids saving the same string multiple times.

Compared to StringPool, it performs fewer allocations but doesn't support refcounting/deletion.
