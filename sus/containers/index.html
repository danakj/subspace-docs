array

auto array(Ts &&... vs)

Defined at line 421 of file array.h

Used to construct an Array<T, N> with the parameters as its values.

Calling array() produces a hint to make an Array<T, N> but does not actually construct Array<T, N>, as the type `T` is not known here.

Note: A marker type is used instead of explicitly constructing an array immediately in order to avoid redundantly having to specify `T` when using the result of `sus::array()` as a function argument or return value.
