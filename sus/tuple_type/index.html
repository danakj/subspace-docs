tuple

auto tuple(Ts &&... vs)

Defined at line 286 of file tuple.h

Used to construct a Tuple<Ts...> with the parameters as its values.

Calling tuple() produces a hint to make a Tuple<Ts...> but does not actually construct Tuple<Ts...>, as the types in `Ts...` are not known here.

Note: A marker type is used instead of explicitly constructing a tuple immediately in order to avoid redundantly having to specify `Ts...` when using the result of `sus::tuple()` as a function argument or return value.
