operator sus::result::State

public State operator sus::result::State()

Defined at line 285 of file result.h

An operator which returns the state of the Result, either #Ok or #Err.

This supports the use of an Result in a `switch()`, allowing it to act as a tagged union between "success" and "error".

# Example

```cpp auto x = Result<int, char>::with(2); switch (x) { case Ok: return sus::move(x).unwrap_unchecked(::sus::marker::unsafe_fn); case Err: return -1; } ```
