class ScopedTrace

Defined at line 2231 of file gtest.h

Causes a trace (including the given source file path and line number, and the given message) to be included in every test failure message generated by code in the scope of the lifetime of an instance of this class. The effect is undone with the destruction of the instance.

The message argument can be anything streamable to std::ostream.

Example: testing::ScopedTrace trace("file.cc", 123, "message");
