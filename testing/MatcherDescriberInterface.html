DescribeTo

public void DescribeTo(::std::ostream * os)

Describes this matcher to an ostream. The function should print a verb phrase that describes the property a value matching this matcher should have. The subject of the verb phrase is the value being matched. For example, the DescribeTo() method of the Gt(7) matcher prints "is greater than 7".

DescribeNegationTo

public void DescribeNegationTo(::std::ostream * os)

Defined at line 126 of file gtest-matchers.h

Describes the negation of this matcher to an ostream. For example, if the description of this matcher is "is greater than 7", the negated description could be "is not greater than 7". You are not required to override this when implementing MatcherInterface, but it is highly advised so that your matcher can produce good error messages.
