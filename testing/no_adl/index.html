FieldsAre

internal::FieldsAreMatcher<typename std::decay<M>::type...> FieldsAre(M &&... matchers)

Defined at line 4818 of file gmock-matchers.h

FieldsAre(matchers...) matches piecewise the fields of compatible structs. These include those that support `get<I>(obj)`, and when structured bindings are enabled any class that supports them. In particular, `std::tuple`, `std::pair`, `std::array` and aggregate types.
