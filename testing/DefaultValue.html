class DefaultValue

Defined at line 573 of file gmock-actions.h

When an unexpected function call is encountered, Google Mock will let it return a default value if the user has specified one for its return type, or if the return type has a built-in default value; otherwise Google Mock won't know what value to return and will have to abort the process.

The DefaultValue<T> class allows a user to specify the default value for a type T that is both copyable and publicly destructible (i.e. anything that can be used as a function return type). The usage is:

// Sets the default value for type T to be foo. DefaultValue<T>::Set(foo);
