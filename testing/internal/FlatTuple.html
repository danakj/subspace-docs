class FlatTuple

Defined at line 1278 of file gtest-internal.h

Analog to std::tuple but with different tradeoffs. This class minimizes the template instantiation depth, thus allowing more elements than std::tuple would. std::tuple has been seen to require an instantiation depth of more than 10x the number of elements in some implementations. FlatTuple and ElemFromList are not recursive and have a fixed depth regardless of T... MakeIndexSequence, on the other hand, it is recursive but with an instantiation depth of O(ln(N)).
