class ValuesInIteratorRangeGenerator

Defined at line 292 of file gtest-param-util.h

Generates values from a pair of STL-style iterators. Used in the ValuesIn() function. The elements are copied from the source range since the source can be located on the stack, and the generator is likely to persist beyond that stack frame.
