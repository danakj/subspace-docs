class NativeArray

Defined at line 1094 of file gtest-internal.h

Adapts a native array to a read-only STL-style container. Instead of the complete STL container concept, this adaptor only implements members useful for Google Mock's container matchers. New members should be added as needed. To simplify the implementation, we only support Element being a raw type (i.e. having no top-level const or reference modifier). It's the client's responsibility to satisfy this requirement. Element can be an array type itself (hence multi-dimensional arrays are supported).
