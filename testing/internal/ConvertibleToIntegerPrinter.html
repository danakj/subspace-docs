public void PrintValue(internal::BiggestInt value, ::std::ostream * os)

Defined at line 249 of file gtest-printers.h

Since T has no << operator or PrintTo() but can be implicitly converted to BiggestInt, we print it as a BiggestInt.

Most likely T is an enum type (either named or unnamed), in which case printing it as an integer is the desired behavior. In case T is not an enum, printing it as an integer is the best we can do given that it has no user-defined printer.
