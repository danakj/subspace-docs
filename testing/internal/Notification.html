class Notification

Defined at line 1235 of file gtest-port.h

Allows a controller thread to pause execution of newly created threads until notified. Instances of this class must be created and destroyed in the controller thread.

This class is only for testing Google Test's own constructs. Do not use it in user tests, either directly or indirectly. TODO(b/203539622): Replace unconditionally with absl::Notification.
