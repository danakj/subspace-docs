class ReferenceOrValueWrapper

Defined at line 1317 of file gmock-spec-builders.h

Wrapper type for generically holding an ordinary value or lvalue reference. If T is not a reference type, it must be copyable or movable. ReferenceOrValueWrapper<T> is movable, and will also be copyable unless T is a move-only value type (which means that it will always be copyable if the current platform does not support move semantics).

The primary template defines handling for values, but function header comments describe the contract for the whole template (including specializations).
