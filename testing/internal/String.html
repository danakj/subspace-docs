CloneCString

public const char * CloneCString(const char * c_str)

Clones a 0-terminated C string, allocating memory using new. The caller is responsible for deleting the return value using delete[]. Returns the cloned string, or NULL if the input is NULL.

This is different from strdup() in string.h, which allocates memory using malloc().
