class Random

Defined at line 870 of file gtest-internal.h

A simple Linear Congruential Generator for generating random numbers with a uniform distribution. Unlike rand() and srand(), it doesn't use global state (and therefore can't interfere with user code). Unlike rand_r(), it's portable. An LCG isn't very random, but it's good enough for our purposes.
