public basic_string UrlEncode(const char * str)

Defined at line 4873 of file gtest.cc

Checks if str contains '=', '&', '%' or '' characters. If yes, replaces them by "%xx" where xx is their hexadecimal value. For example, replaces "=" with "%3D". This algorithm is O(strlen(str)) in both time and space -- important as the input str may contain an arbitrarily long test failure message and stack trace.

Escapes '=', '&', '%', and '' characters in str as "%xx".
