class ExpectationBase

Defined at line 682 of file gmock-spec-builders.h

Base class for expectations.

Base class for implementing expectations.

There are two reasons for having a type-agnostic base class for Expectation:

1. We need to store collections of expectations of different types (e.g. all pre-requisites of a particular expectation, all expectations in a sequence). Therefore these expectation objects must share a common base class.

2. We can avoid binary code bloat by moving methods not depending on the template argument of Expectation to the base class.

This class is internal and mustn't be used by user code directly.
