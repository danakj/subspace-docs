class ReturnAction

Defined at line 1089 of file gmock-actions.h

A specialization of ReturnAction<R> when R is ByMoveWrapper<T> for some T.

This version applies the type system-defeating hack of moving from T even in the const call operator, checking at runtime that it isn't called more than once, since the user has declared their intent to do so by using ByMove.
