struct InvokeWithoutArgsAction

Defined at line 938 of file gmock-actions.h

Implements the InvokeWithoutArgs(f) action. The template argument FunctionImpl is the implementation type of f, which can be either a function pointer or a functor. InvokeWithoutArgs(f) can be used as an Action<F> as long as f's type is compatible with F.
