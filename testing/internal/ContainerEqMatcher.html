class ContainerEqMatcher

Defined at line 2407 of file gmock-matchers.h

Implements an equality matcher for any STL-style container whose elements support ==. This matcher is like Eq(), but its failure explanations provide more detailed information that is useful when the container is used as a set. The failure message reports elements that are in one of the operands but not the other. The failure messages do not report duplicate or out-of-order elements in the containers (which don't properly matter to sets, but can occur if the containers are vectors or lists, for example).

Uses the container's const_iterator, value_type, operator ==, begin(), and end().
