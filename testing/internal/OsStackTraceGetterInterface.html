CurrentStackTrace

public basic_string CurrentStackTrace(int max_depth, int skip_count)

Returns the current OS stack trace as an std::string. Parameters:

max_depth - the maximum number of stack frames to be included in the trace. skip_count - the number of top frames to be skipped; doesn't count against max_depth.

UponLeavingGTest

public void UponLeavingGTest()

UponLeavingGTest() should be called immediately before Google Test calls user code. It saves some information about the current stack that CurrentStackTrace() will use to find and hide Google Test stack frames.
