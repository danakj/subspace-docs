class KeyMatcherImpl

Defined at line 2949 of file gmock-matchers.h

Implements Key(inner_matcher) for the given argument pair type. Key(inner_matcher) matches an std::pair whose 'first' field matches inner_matcher. For example, Contains(Key(Ge(5))) can be used to match an std::map that contains at least one element whose key is >= 5.
