class BuiltInDefaultValue

Defined at line 195 of file gmock-actions.h

BuiltInDefaultValue<T>::Get() returns the "built-in" default value for type T, which is NULL when T is a raw pointer type, 0 when T is a numeric type, false when T is bool, or "" when T is string or std::string. In addition, in C++11 and above, it turns a default-constructed T value if T is default constructible. For any other type T, the built-in default T value is undefined, and the function will abort the process.
