class VariadicMatcher

Defined at line 1304 of file gmock-matchers.h

VariadicMatcher is used for the variadic implementation of AllOf(m_1, m_2, ...) and AnyOf(m_1, m_2, ...). CombiningMatcher<T> is used to recursively combine the provided matchers (of type Args...).
