struct IsHashTable

Defined at line 964 of file gtest-internal.h

Trait to detect whether a type T is a hash table. The heuristic used is that the type contains an inner type `hasher` and does not contain an inner type `reverse_iterator`. If the container is iterable in reverse, then order might actually matter.
