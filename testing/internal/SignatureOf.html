struct SignatureOf

The SignatureOf<F> struct is a meta-function returning function signaturecorresponding to the provided F argument.

It makes use of MockFunction easier by allowing it to accept more F argumentsthan just function signatures.

Specializations provided here cover a signature type itself and any templatethat can be parameterized with a signature, including std::function andboost::function.
