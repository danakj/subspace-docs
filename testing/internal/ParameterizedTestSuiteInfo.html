class ParameterizedTestSuiteInfo

Defined at line 496 of file gtest-param-util.h

INTERNAL IMPLEMENTATION - DO NOT USE IN USER CODE.

ParameterizedTestSuiteInfo accumulates tests obtained from TEST_P macro invocations for a particular test suite and generators obtained from INSTANTIATE_TEST_SUITE_P macro invocations for that test suite. It registers tests with all values generated by all generators when asked.
