class ParamGenerator

Defined at line 183 of file gtest-param-util.h

Wraps ParamGeneratorInterface<T> and provides general generator syntax compatible with the STL Container concept. This class implements copy initialization semantics and the contained ParamGeneratorInterface<T> instance is shared among all copies of the original object. This is possible because that instance is immutable.
