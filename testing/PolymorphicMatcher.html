class PolymorphicMatcher

Defined at line 633 of file gtest-matchers.h

The PolymorphicMatcher class template makes it easy to implement a polymorphic matcher (i.e. a matcher that can match values of more than one type, e.g. Eq(n) and NotNull()).

To define a polymorphic matcher, a user should provide an Impl class that has a DescribeTo() method and a DescribeNegationTo() method, and define a member function (or member function template)

bool MatchAndExplain(const Value& value, MatchResultListener* listener) const;

See the definition of NotNull() for a complete example.
