class UnitTest

Defined at line 1274 of file gtest.h

A UnitTest consists of a vector of TestSuites.

This is a singleton class. The only instance of UnitTest is created when UnitTest::GetInstance() is first called. This instance is never deleted.

UnitTest is not copyable.

This class is thread-safe as long as the methods are called according to their specification.
