class Action

Defined at line 728 of file gmock-actions.h

An Action<R(Args...)> is a copyable and IMMUTABLE (except by assignment) object that represents an action to be taken when a mock function of type R(Args...) is called. The implementation of Action<T> is just a std::shared_ptr to const ActionInterface<T>. Don't inherit from Action! You can view an object implementing ActionInterface<F> as a concrete action (including its current state), and an Action<F> object as a handle to it.

Perform

public Result Perform(ArgumentTuple args)

Defined at line 789 of file gmock-actions.h

Performs the action. Note that this method is const even though the corresponding method in ActionInterface is not. The reason is that a const Action<F> means that it cannot be re-bound to another concrete action, not that the concrete action it binds to cannot change state. (Think of the difference between a const pointer and a pointer to const.)
