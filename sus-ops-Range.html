Struct sus::ops::Range
struct Range
{ ... };

A (half-open) range bounded inclusively below and exclusively above (start..end).

The range start..end contains all values with start <= x < end. It is empty if start >= end.

In most cases a Range is passed as an argument to a function or operator[] call, and it can be constructed simply as {start, end}.

Data Members
T end

The end of the range, exclusive of the given value.

T start

The beginning of the range, inclusive of the given value.

Methods
bool contains (T) const

Returns true if item is contained in the range.

bool is_empty () const

Returns true if the range contains no items.

The range is empty if either side is incomparable, such as f32::NAN.
